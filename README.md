# Alcohol Free
나날이 성장 하고 있는 주류 시장은 온라인에서의 판매가 제한적이기 때문에, 판매가 아닌 판매 중계를 하는 사이트를 만들어 보고자 기획하게 되었음

## 프로젝트 개요
- 프로젝트 명 : Alcohol Free
- 프로젝트 목표 : 주류 판매와 관련된 다양한 제품을 소비자에게 판매 중계
- 기간 : 2023/10/23 ~ 2023/11/03
